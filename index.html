Security
Enterprise-Grade Protection
A full security audit was completed in February 2026. Here's what protects the platform.
🔐

AES-256 Encryption

Tax File Numbers encrypted at rest with IV, masked in responses

🛡

SQL Injection Prevention

All queries use parameterized $N placeholders, validated inputs

🛑

Rate Limiting

500 req/15min general, 20 req/15min on auth endpoints

🌐

CORS & Helmet

Strict origin allowlist, security headers via Helmet

👀

Audit Logging

TFN access logs, email logs, sync logs — full traceability

😶

Data Masking

Bank details show last 4 digits only, TFN shows last 3
